Installation
============

Stable release
--------------

To install Test Pyckage GitHub, run this command in your terminal:

.. code-block:: console

    $ pip install test_pyckage_github

This is the preferred method to install Test Pyckage GitHub, as it will always install the most recent stable release.


From sources
------------

The sources for Test Pyckage GitHub can be downloaded from the `Github repo <https://github.com/KnightConan/test-pyckage-github/settings/secrets/actions>`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone https://github.com/KnightConan/test-pyckage-github/settings/secrets/actions.git

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install

or

.. code-block:: console

    $ pip install .
